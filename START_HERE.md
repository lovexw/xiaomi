## 🚀 立即部署（3分钟）

### 步骤 1️⃣：打开 Cloudflare
访问：**https://dash.cloudflare.com/**
（如无账号，免费注册）

### 步骤 2️⃣：创建 Pages 项目
1. 点击左侧 **"Pages"**
2. 点击 **"创建项目"**
3. 选择 **"直接上传"**

### 步骤 3️⃣：上传文件
将以下文件全部上传：
```
✓ index.html
✓ styles.css
✓ script.js
✓ _headers
✓ _redirects
✓ robots.txt
✓ sitemap.xml
```

### 步骤 4️⃣：部署
- 项目名称：输入 `aohan-millet`
- 点击 **"部署站点"**

### 步骤 5️⃣：完成！ 🎉
您的网站地址：`https://aohan-millet.pages.dev`
