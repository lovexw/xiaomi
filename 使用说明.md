## 🚀 部署指南

### 方式一：直接上传（最简单）⭐

**适合人群**：不熟悉Git或想快速上线

1. **准备文件**
   - 确保有 `index.html`, `styles.css`, `script.js` 三个文件

2. **登录 Cloudflare**
   - 访问：https://dash.cloudflare.com/
   - 如无账号，免费注册一个

3. **创建项目**
   - 点击左侧 "Pages"
   - 点击 "创建项目"
   - 选择 "直接上传"

4. **上传文件**
   - 拖拽所有文件到上传区
   - 或点击"选择文件"按钮

5. **配置**
   - 项目名称：`aohan-millet`（可自定义）

6. **部署**
   - 点击"部署站点"
   - 等待1-2分钟

7. **完成**
   - 获得网址：`https://aohan-millet.pages.dev`
   - 立即访问查看效果！

### 方式二：Git自动部署（推荐）

**适合人群**：开发者或需要频繁更新

**优点**：
- ✅ 自动部署，推送代码即更新
- ✅ 版本控制，可回退
- ✅ 多人协作

**步骤**：

1. **创建Git仓库**（以GitHub为例）
   ```bash
   # 本地初始化
   git init
   git add .
   git commit -m "初始提交"
   
   # 推送到GitHub
   git remote add origin https://github.com/你的用户名/aohan-millet.git
   git push -u origin main
   ```

2. **连接Cloudflare**
   - 登录 Cloudflare Dashboard
   - Pages → 创建项目 → 连接Git
   - 选择GitHub仓库
   - 配置：
     - 构建命令：留空
     - 输出目录：/
   - 保存并部署

3. **自动更新**
   - 修改文件后：
   ```bash
   git add .
   git commit -m "更新内容"
   git push
   ```
   - Cloudflare自动重新部署

## 🎨 自定义修改

### 修改文本内容

1. **打开 `index.html` 文件**
   - 使用任意文本编辑器（记事本、VSCode等）

2. **找到要修改的部分**
   - 使用查找功能（Ctrl+F）搜索关键词

3. **修改文本**
   - 直接编辑HTML标签内的文字
   - 保存文件

4. **预览效果**
   - 双击 `index.html` 在浏览器中打开
   - 或使用本地服务器

### 修改颜色主题

打开 `styles.css`，找到顶部的颜色变量：

```css
:root {
    --primary-color: #D4A574;      /* 主色调 */
    --primary-dark: #B8904E;       /* 深色 */
    --primary-light: #E8C9A3;      /* 浅色 */
    --secondary-color: #8B7355;    /* 辅助色 */
    --accent-color: #F5DEB3;       /* 强调色 */
}
```

修改这些颜色值即可改变整个网站的配色。

### 修改字体

在 `index.html` 的 `<head>` 部分修改 Google Fonts 引用：

```html
<link href="https://fonts.googleapis.com/css2?family=你的字体&display=swap" rel="stylesheet">
```

然后在 `styles.css` 中修改：

```css
body {
    font-family: '你的字体', sans-serif;
}
```

### 添加新内容

1. **找到合适的位置**
   - 在 `index.html` 中找到要插入的 `<section>`

2. **复制类似的结构**
   - 复制现有的卡片或模块代码

3. **修改内容**
   - 更改文字、图标等

4. **保存并测试**

### 添加图片

1. **准备图片**
   - 将图片放在项目文件夹中

2. **在HTML中引用**
   ```html
   <img src="你的图片.jpg" alt="描述">
   ```

3. **优化图片**
   - 压缩图片大小
   - 使用 WebP 格式

## 🔧 常见问题

### Q1: 如何在本地预览网站？

**A1**: 有多种方法：

**方法1：直接打开**
- 双击 `index.html` 文件

**方法2：使用Python**
```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

**方法3：使用VSCode**
- 安装 "Live Server" 插件
- 右键点击 index.html
- 选择 "Open with Live Server"

### Q2: 修改后如何更新到网站？

**A2**: 取决于部署方式：

**直接上传方式**：
1. 登录 Cloudflare Pages
2. 进入项目
3. 上传新版本文件

**Git方式**：
```bash
git add .
git commit -m "更新内容"
git push
```
自动部署

### Q3: 网站加载慢怎么办？

**A3**: 检查以下几点：
- ✅ Cloudflare CDN 已启用（自动）
- ✅ 图片已优化（如果有）
- ✅ `_headers` 文件已上传（开启缓存）

### Q4: 如何添加自己的域名？

**A4**: 
1. 进入 Cloudflare Pages 项目
2. 点击"自定义域"
3. 添加您的域名
4. 配置DNS（Cloudflare会提供指引）
5. 等待SSL证书生效（自动）

### Q5: 移动端显示不正常？

**A5**: 
- 清除浏览器缓存
- 确保 `viewport` meta标签存在
- 测试不同设备

### Q6: 动画效果不显示？

**A6**: 
- 确保 `script.js` 正确加载
- 检查浏览器控制台是否有错误
- 更新浏览器到最新版本

### Q7: 如何修改联系方式？

**A7**: 
1. 打开 `index.html`
2. 搜索 "联系" 或 "contact"
3. 修改相关内容
4. 保存并重新部署

### Q8: 可以添加评论功能吗？

**A8**: 
可以，但需要第三方服务：
- Disqus
- Gitalk
- Utterances

在 HTML 中添加对应的代码即可。

### Q9: 如何添加统计分析？

**A9**: 

**方法1：使用 Cloudflare Web Analytics（推荐）**
- 在 Pages 设置中启用
- 免费且隐私友好

**方法2：Google Analytics**
- 在 `</head>` 前添加跟踪代码

### Q10: 网站被墙了怎么办？

**A10**: 
- Cloudflare 提供的 `.pages.dev` 域名在中国可访问
- 如使用自定义域名，确保备案
- 考虑使用国内CDN
